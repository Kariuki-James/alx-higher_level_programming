=========================
The '1-my_list' module
=========================
Defines a Class ``MyList`` that inherits from ``list``


USAGE
======

>>> MyList = __import__('1-my_list').MyList
>>>


:: Instatiation
----------------------------------
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>>

:: ``print_sorted`` method
----------------------------------
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

>>> my_list = MyList([50, 40, 30, 20, 10])
>>> my_list.print_sorted()
[10, 20, 30, 40, 50]
